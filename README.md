The Cafe Management System is a full-stack web application designed to streamline and automate day-to-day café operations. The system enables efficient management of menu items, customer orders, billing, and data storage through a simple and intuitive interface. The application is developed using HTML and CSS for the front-end, Python Flask for the back-end, and SQLite for persistent data management.

Key Features:

Order Management:
Allows staff to take and modify orders, manage quantities, calculate totals, and generate real-time bills.

Menu Management:
Admin can add, update, or remove menu items, including item names, categories, and pricing.

Billing System:
Automatically calculates total amounts, taxes (if applied), and generates final bills for customers.

User-Friendly Interface:
Responsive UI built using HTML and CSS to ensure easy navigation and efficient workflow for café staff.

Data Storage and Retrieval:
Customer orders, menu data, and billing information are stored securely in SQLite and retrieved efficiently through Flask routes.

Role-Based Access (optional):
Admin access for menu and system management, while staff access focuses on order processing.

Technology Stack:

Front-End: HTML, CSS

Back-End: Python Flask Framework

Database: SQLite

Tools: SQL queries, Jinja templates, Flask routing and session handling
